# Team Task Manager

A full-stack Team Task Management application built using React, Node.js, Express, Prisma, and PostgreSQL.

## Features

- User Authentication (JWT)
- Signup & Login
- Role-Based Access Control
- Project Management
- Task Management
- Dashboard Analytics
- Responsive UI
- REST API Integration

---

# Tech Stack

## Frontend
- React.js
- Vite
- React Router DOM
- Axios
- CSS

## Backend
- Node.js
- Express.js
- Prisma ORM
- PostgreSQL
- JWT Authentication
- bcryptjs

## Deployment
- Frontend: Vercel
- Backend: Render
- Database: PostgreSQL
